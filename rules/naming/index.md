Rule kinds in the **Naming** family. Each entry below has its own page with options, an example, and any auto-fix support.

- [`filename_case`](/docs/rules/naming/filename_case/) — Basename (stem only or full) matches a case convention: `snake`, `kebab`, `pascal`, `camel`, `screaming-snake`, `flat`, `lower`, `upper`.
- [`filename_regex`](/docs/rules/naming/filename_regex/) — Basename matches a regex.
